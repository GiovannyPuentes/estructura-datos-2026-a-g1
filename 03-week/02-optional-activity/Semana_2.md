### 🔹 1. Array unidimensional

Supongamos que estamos desarrollando una aplicación para registrar las edades de 5 estudiantes.

python
edades = [16, 18, 17, 15, 19]

### 📌 ¿Qué representa cada posición?

- edades[0] → Edad del Estudiante 1 (16 años)
- edades[1] → Edad del Estudiante 2 (18 años)
- edades[2] → Edad del Estudiante 3 (17 años)
- edades[3] → Edad del Estudiante 4 (15 años)
- edades[4] → Edad del Estudiante 5 (19 años)

Cada posición representa la edad de un estudiante diferente.

### 🔹 2. Matriz (mínimo 2x3)

Ahora registraremos las notas de dos estudiantes en tres materias.

python
notas = [
 [4.3, 3.9, 4.1],
 [3.7, 4.6, 4.2]
]

📊 La matriz tiene:

- 2 filas (2 estudiantes)
- 3 columnas (3 materias)

### 📌 ¿Qué representa cada fila?

- notas[0] → Notas del Estudiante 1
 - Matemáticas: 4.3
 - Español: 3.9
 - Informática: 4.1

- notas[1] → Notas del Estudiante 2
 - Matemáticas: 3.7
 - Español: 4.6
 - Informática: 4.2

Cada fila representa todas las notas de un estudiante.

### 🔹 3. Operación que se puede realizar

Una operación que podemos realizar es *calcular el promedio de notas por estudiante*.

Ejemplo:

Promedio Estudiante 1:
(4.3 + 3.9 + 4.1) / 3 = 4.1

Promedio Estudiante 2:
(3.7 + 4.6 + 4.2) / 3 = 4.16

📌 Esta operación permite analizar el rendimiento académico de cada estudiante.

## 🧠 Conclusión

Los arrays permiten almacenar datos en una sola dimensión, mientras que las matrices organizan información en filas y columnas.

Estas estructuras son fundamentales para organizar datos de forma clara y eficiente en cualquier aplicación.
